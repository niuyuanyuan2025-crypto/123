# Automated Content Creation Tool

## Overview
This project is an automated content creation tool designed to fetch trending topics, select them, and generate AI-driven content. This tool will cater to content creators, marketers, and podcasters who wish to automate their content generation processes.

## Features
- **Trending Topics Fetching**: The tool retrieves the latest trending topics from various sources, ensuring that the content generated is relevant and up-to-date.
- **Topic Selection**: Users can specify keywords or categories to filter the trending topics, making it easier to select content that aligns with their niche.
- **AI Content Generation**: Utilizes advanced AI models to generate high-quality content based on selected topics. This includes blog posts, articles, and social media posts.
- **Podcast Scripts Creation**: Automatically generates scripts for podcasts based on the selected topics, providing a comprehensive narrative that flows logically.
- **Public Posts Generation**: Creates engaging public posts tailored for social media platforms, enhancing user interaction and reach.

## Installation

```bash
git clone https://github.com/niuyuanyuan2025-crypto/123.git
cd 123
pip install -r requirements.txt
```

## Usage
1. **Fetch Trending Topics**: Run the command `python fetch_topics.py` to retrieve trending topics.
2. **Select Topics**: Use the command `python select_topics.py -k "your_keyword"` to filter topics based on your interests.
3. **Generate Content**: Run `python generate_content.py -t "selected_topic"` to generate content.
4. **Create Podcast Script**: Use `python create_script.py -t "selected_topic"` for podcast scripts.
5. **Generate Public Post**: Run `python create_post.py -t "selected_topic"` for social media posts.
